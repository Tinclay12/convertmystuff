# ConvertMyStuff Cursor Rules

Copy the `.cursor` folder into the root of your ConvertMyStuff project.

Recommended location:

```text
convertmystuff/
  .cursor/
    rules/
      001-project-mission.md
      002-architecture-rules.md
      ...
  docs/
    CURSOR_START_HERE.md
    01-product-brief.md
    ...
```

This package includes both `.md` and `.mdc` versions of the rules. Cursor commonly uses `.mdc` for project rules, but the `.md` copies are included because they are easier to review and edit.

After copying the folder, open `.cursor/rules/007-initial-cursor-prompt.md` and paste that prompt into Cursor.
